1. Specify a schema name in app.config file in the VS project.
2. Update the SQL connection string app.config file.
3. Update the SQL script file 'Zoom script.sql' in SQLScripts folder with your database name and schema name.
3. Run the SQL script file using SQL Management Studio.
4. Run the application using Visual Studio by pressing F5. 
5. If you have an error message 'couldn't process file due to being in the internet...', locate the file 'Form1.resx' in the File Explorer. Unlock the file through its properties.
